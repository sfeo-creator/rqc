Bienvenue sur la page d'accueil du rQC

Le rQC c'est le prix de l'alcool pur d'une bouteille rapporté à un litre.
Il permet de savoir à quel prix est la race que je me mets.

Pour utiliser l'application il faut d'abord que vous importiez votre base de données en cliquant en dessous sur "Choisir un fichier" puis "Importer les données".
Une fois votre base de données importée, vous pourrez accéder aux fonctionnalités de l'application. Il est possible de visualiser les données dans le menu "Affichage des données" et
d'en supprimer depuis ce menu. Le menu "enregistrement des données" vous permet de ajouter de nouvelles données à la base de données.
Une fois votre base de données complétée, vous pouvez l'exporter via le bouton "Exporter les données". Le fichier .JSON que vous téléchargerez pourra être partagé à vos amis ou sur le drive afin qu'une nouvelles
base de données enrichie par les données de tout le monde soit créée.
Celle-ci devrait paraitre tous les mois ou 2 à 3 semaines.
